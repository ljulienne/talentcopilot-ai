# TalentCopilot Next — Release 0.5.3

## Decision Timeline & Progress Tracking

This release turns the AI Decision Queue into a traceable execution workflow.

### Capabilities

- Human-controlled status transitions.
- Proposed, Accepted, In progress, Completed and Dismissed states.
- Immutable decision event history.
- Completion and active-decision metrics.
- Notes and actor attribution.
- CSV timeline export.
- Streamlit session persistence for the active analysis.

### Product principle

TalentCopilot proposes. Humans validate, execute and remain accountable.
