# Integration Guide — Release 6.0A

1. Start from GitHub HEAD `5d19caf6` or verify the installer base check.
2. Copy the release payload over the repository root.
3. Run the targeted Release 6.0A tests.
4. Run the complete pytest suite.
5. Launch Streamlit and open **Recruitment Workspace**.

The navigation label remains unchanged for compatibility. Its renderer now delegates to `talentcopilot.recruitment.mission.workspace.render_recruitment_mission_workspace`.
