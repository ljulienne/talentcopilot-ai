# TalentCopilot Release 5.1 — Comparative Ranking Engine

Release 5.1 adds a comparative decision layer above Mission Fit Engine v2.

## Capabilities

- differentiates high-fit candidates by role seniority and scope;
- evaluates geographic, leadership, commercial and industry depth;
- analyses career progression;
- preserves Mission Fit as the primary eligibility signal;
- prevents comparative scope from rescuing a low-fit candidate;
- stores recruiter-readable differentiators and validation points;
- enriches the official score breakdown with comparative dimensions.

## Reference benchmark

For the Senior Sales Manager APAC benchmark, the expected order is:

1. John Anderson
2. Sarah Lim
3. David Nguyen
4. Emily Carter
5. Michael Brown

The benchmark is based on role and scope evidence, never on candidate-name rules.
