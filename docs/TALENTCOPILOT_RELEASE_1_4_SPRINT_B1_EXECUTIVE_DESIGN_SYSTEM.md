# TalentCopilot Release 1.4 — Sprint B1

## Executive Design System

This sprint introduces import-safe, reusable Streamlit components for executive experiences.

### Components

- Executive health card
- Metric and confidence cards
- Priority card
- Evidence card
- Recommendation card
- Decision timeline
- Section headers
- Central executive visual tokens

The package uses lazy Streamlit imports so non-UI tests and command-line tools remain safe.
