1. Install on the exact snapshot commit.
2. Run targeted tests.
3. Run the complete pytest suite.
4. Open Recruitment Workspace with no active session.
5. Load sample data or upload a real JD and CVs.

7. Confirm the workflow rail progresses from Mission to Decision.
8. Confirm all existing workspace tabs still render.
