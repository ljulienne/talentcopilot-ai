# TalentCopilot Enterprise Blueprint

This folder contains the architecture reference for TalentCopilot Enterprise.

The Blueprint defines the long-term product architecture, business rules, decision intelligence framework and roadmap.

## Why this Blueprint exists

TalentCopilot is not only an ATS or a CV matching tool.

TalentCopilot is an AI Hiring Decision Platform designed to help organizations make recruitment decisions that are:

- explainable;
- collaborative;
- evidence-based;
- auditable;
- human-controlled.
