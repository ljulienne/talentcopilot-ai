# 14 — Executive Intelligence Engine

## Purpose

Executive Intelligence converts decision signals into stakeholder-ready summaries.

It answers:

> How should this recommendation be explained to each audience?

## Audiences

- Recruiter
- Hiring Manager
- HR Director
- Executive Committee

## Design rules

1. Executive Intelligence does not recalculate scores.
2. It must preserve the recommendation and rationale from Recommendation Intelligence.
3. It must clearly separate candidate fit, risk, confidence and budget feasibility.
4. It must be concise enough for leadership but detailed enough for audit.
5. It must reference the Decision Trace.

## Outputs

- Recruiter Summary
- Hiring Manager Summary
- HR Director Summary
- Executive Summary
- Markdown Decision Summary
