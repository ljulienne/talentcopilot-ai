# Talent Knowledge Base

The Talent Knowledge Base is the business knowledge layer of TalentCopilot.

It contains:

- Skills
- Capabilities
- Tools
- Certifications
- Languages
- Job families
- Industries

Each knowledge item uses a stable identifier such as:

- SK-000001 for skills
- CAP-000001 for capabilities
- TOOL-000001 for tools

This allows the platform to evolve without breaking relationships.
