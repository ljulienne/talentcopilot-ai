# Release 7.2.0 — Candidate Intelligence Enrichment

## Scope

This release turns the five Candidate Intelligence tabs into complementary decision views while preserving the canonical Mission Fit score and canonical rank.

## Changes

- Overview now includes a candidate-specific recommendation, rationale, principal strengths, material-risk count and next action.
- Skills now use distinct deterministic evidence assessments per capability instead of one shared level.
- Evidence now exposes requirement, source, ownership, outcome, confidence and evidence type.
- Risks now distinguish confirmed risks, probable risks and validation points, with severity, evidence basis and interview question.
- Interview Focus now includes priority validation areas, evidence-based probes, positive signals and warning signals.
- CandidateWorkspaceReport remains the single presentation source shared by all five tabs.

## Stability

The release does not recalculate or mutate official matching scores, official ranks, candidate identities or source-of-truth analyses.
