# Release 6.1A — Recruiter Intelligence Foundation

## Purpose
Add an evidence-grounded recruiter reasoning layer above Mission Fit, Comparative Ranking and Calibrated Scoring.

## Added
- `talentcopilot.recruiter_intelligence`
- job persona detection
- Candidate DNA archetypes
- strategic qualitative dimensions
- decisive strengths, material gaps and interview focus
- recruiter summary and confidence
- pipeline metadata integration

## Contract
The engine does **not** replace or mutate the official public matching score or rank. Mission Fit remains the source of truth. Recruiter Intelligence interprets the evidence and stores a structured assessment in profile metadata.

## Version
`recruiter-intelligence-v1.0`
