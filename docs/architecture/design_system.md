# TalentCopilot Design System

## Foundations

### Brand
TalentCopilot Enterprise uses a modern, sober and enterprise-oriented visual language.

### Colors
- Primary: #2563EB
- Secondary: #06B6D4
- AI: #7C3AED
- Success: #22C55E
- Warning: #F59E0B
- Danger: #EF4444
- Background: #F8FAFC
- Surface: #FFFFFF
- Text: #0F172A

### Logo
The logo uses a custom geometric mark representing talent, connection and decision intelligence.

### Component Rules
- Components must be reusable.
- Components should not know business logic.
- AI recommendations must show reason, confidence and next action.
- Cards must use consistent spacing, radius and hierarchy.
