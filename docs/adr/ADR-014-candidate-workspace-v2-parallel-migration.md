# ADR-014 — Candidate Workspace v2 parallel migration

## Status

Accepted

## Context

The existing Candidate Workspace is stable and demo-ready.

Replacing it immediately would be risky.

## Decision

Create a parallel Candidate Workspace v2 powered by Decision Core.

## Consequences

- Current demo remains stable.
- New architecture becomes visible.
- Full migration can happen once the v2 workspace is validated.
