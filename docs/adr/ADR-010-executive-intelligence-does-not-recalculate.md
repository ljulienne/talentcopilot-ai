# ADR-010 — Executive Intelligence does not recalculate decisions

## Status

Accepted

## Context

Executive summaries must be consistent with the Decision Core.

If summaries recalculate or reinterpret scores independently, inconsistencies can reappear.

## Decision

Executive Intelligence only explains outputs from other Decision Core engines.

## Consequences

- Summaries remain aligned with recommendation logic.
- Reporting becomes stakeholder-ready and auditable.
- Decision Trace stays the source of reasoning history.
