# ADR-019 — Real Ranking uses Decision Core outputs

## Status

Accepted

## Context

TalentCopilot must rank candidates based on real extracted data.

Ranking directly from raw text would duplicate logic and create inconsistency.

## Decision

Real Ranking uses CandidateDecisionProfiles generated by Decision Core.

## Consequences

- Ranking remains explainable.
- Recommendation logic is reused.
- Future UI pages can consume the same ranking output.
