# Architecture

TalentCopilot is built around a modular architecture.

## Main flow

Job Description PDF
→ PDF Parser
→ Job Builder
→ Talent Knowledge Base
→ Candidate CV PDFs
→ Candidate Builder
→ Matching Engine
→ Recruiter Dashboard

## Key components

- Streamlit UI
- PDF Parser
- AI Provider
- Talent Knowledge Base
- Matching Engine
- Capability Engine
- Decision Engine
