# Integration Guide — Release 6.2C

The real ranking pipeline now calls `DecisionRankingPolicy.evaluate()` after Mission Fit, Recruiter Intelligence and Career Intelligence.

Consumers must use:
- `mission_fit_score` and `mission_rank` for objective matching;
- `decision_score`, `decision_rank` and `interview_priority` for recommended interview order.

The source-of-truth snapshot is version `recruitment-sot-v1.1`. Existing session snapshots should be regenerated by rerunning the recruitment analysis after deployment.
